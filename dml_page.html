DML Commands

DML stands for Data Manipulation Language. It is a subset of SQL (Structured Query Language) used to manage data within database tables. DML commands are primarily concerned with inserting, updating, deleting, and modifying data in the database tables. Here are the main DML commands:

  1. INSERT: Adds new rows of data into a table.
    Example:
    INSERT INTO table_name (column1, column2) VALUES (value1, value2);
  2. UPDATE: Modifies existing data in a table.
    Example:
    UPDATE table_name SET column1 = value1, column2 = value2 WHERE condition;
  3. DELETE: Removes rows of data from a table.
    Example:
    DELETE FROM table_name WHERE condition;
  4. MERGE: Performs insert, update, or delete operations on a target table based on the results of a join with a source table.
    Example:
    MERGE INTO target_table USING source_table ON condition WHEN MATCHED THEN UPDATE SET column1 = value1 WHEN NOT MATCHED THEN INSERT (column1, column2) VALUES (value1, value2);

DML commands are essential for maintaining the integrity and consistency of data within a database. They allow for the addition, modification, and removal of data, which are fundamental operations in database management.

DQL Commands

Certainly! Here are the main commands of S4U (Study for you) in SQL:

  1. SELECT: Retrieves data from one or more tables based on specified columns and conditions.
    Example:
    SELECT column1, column2 FROM table_name WHERE condition;
  2. FROM: Specifies the tables from which to retrieve data.
    Example:
    SELECT column1, column2 FROM table_name;
  3. WHERE: Filters data based on specified conditions.
    Example:
    SELECT column1, column2 FROM table_name WHERE condition;
  4. GROUP BY: Groups rows that have the same values into summary rows.
    Example:
    SELECT column1, COUNT(*) FROM table_name GROUP BY column1;
  5. HAVING: Filters groups based on specified conditions.
    Example:
    SELECT column1, COUNT(*) FROM table_name GROUP BY column1 HAVING COUNT(*) > 1;
  6. ORDER BY: Sorts the result set based on specified columns and sort order.
    Example:
    SELECT column1, column2 FROM table_name ORDER BY column1 ASC;
  7. DISTINCT: Returns unique values from a specified column in the result set.
    Example:
    SELECT DISTINCT column1 FROM table_name;

These are the primary commands used in S4U (Study for you) SQL for querying and retrieving data from a database.
